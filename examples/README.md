# Example Files

For testing/education/research purposes, the following files were obtained online directly from Microsoft (i.e. they were not obtained from game discs, and do not contain any game data).

## XEX Files

The following XEX files were provided freely by Microsoft, and I have included them in this repository for convenience and backup. The XEX file hashes are also available here to ensure they are untampered.

### Original Xbox Game Support

Microsoft provided direct download of an XEX file as part of allowing Xbox 360 consoles to play Original Xbox games [here](https://web.archive.org/web/20090906002445/http://www.xbox.com/en-US/games/backwardscompatibility.htm). Users were directed to download the XEX file to be burned onto a disc. Versions include [November 2005](https://web.archive.org/web/20070311061307/http://assets.xbox.com/en-us/backwardcompatibility/default.zip), [December 2005](https://web.archive.org/web/20070315222633/http://download.microsoft.com/download/5/8/a/58a10dbc-9445-4eed-83d5-22f8ad8ee855/default.zip), [June 2006](https://web.archive.org/web/20060901032439/http://download.microsoft.com/download/2/0/9/209a035c-b98a-4ad1-9073-2ebb08182c29/default.zip), [August 2006](https://web.archive.org/web/20061113045821/http://download.microsoft.com/download/5/0/8/508cfd6d-ee87-4ceb-837d-4741a99d4155/default.zip), [December 2006](https://web.archive.org/web/20070103133051/http://download.microsoft.com:80/download/d/1/8/d181ee58-de70-4484-936b-0e9161ccd6b2/backcompat_12-2006.zip), [January 2007](https://web.archive.org/web/20070314014656/http://download.microsoft.com/download/5/0/8/508cfd6d-ee87-4ceb-837d-4741a99d4155/BackCompat_01-2007.zip), [April 2007](https://web.archive.org/web/20110401221354/http://download.microsoft.com/download/5/0/8/508cfd6d-ee87-4ceb-837d-4741a99d4155/BackCompat_04-2007.zip), [July 2007](https://web.archive.org/web/20121029215512/http://download.microsoft.com/download/5/0/8/508cfd6d-ee87-4ceb-837d-4741a99d4155/BackCompat_07-2007.zip), and [November 2007](https://web.archive.org/web/20090601195157/http://download.microsoft.com/download/d/1/8/d181ee58-de70-4484-936b-0e9161ccd6b2/BackCompat_11-2007.zip).

### HD DVD Player

Microsoft provided an update to the Xbox 360 Console for playing HD DVD discs with their external player [here](https://web.archive.org/web/20090805081318/http://www.xbox.com/en-us/hardware/x/xbox360hddvdplayer/download.htm#disc). Versions include [October 2006](https://web.archive.org/web/20070304231706/http://assets.xbox.com/en-us/hardware/HD_DVD_10-2006.zip), [January 2007](https://web.archive.org/web/20070212052009/http://download.microsoft.com/download/d/1/8/d181ee58-de70-4484-936b-0e9161ccd6b2/HD_DVD_01-2007.zip), [May 2007](https://web.archive.org/web/20071012204343/http://download.microsoft.com/download/d/1/8/d181ee58-de70-4484-936b-0e9161ccd6b2/HD_DVD_05-2007.zip), and [December 2007](https://web.archive.org/web/20130514173410/http://assets.xbox.com/en-us/hardware/hddvd/HD_DVD_12-2007.zip). Another unarchived update is [`HD_DVD_112006.zip`](https://web.archive.org/web/20061126052212/http://www.xbox.com/NR/rdonlyres/29F65DE4-3AC9-4140-AB89-F3F9952D14F1/0/HD_DVD_112006.zip).

### Media Center Extender

The Windows PC installer for the Media Center Extender contains a "XboxMcx.xex" file that was executed by the Xbox 360 console when being used as an extender. The file was originally provided by Microsoft as part of the Windows XP Media Center Edition 2005 – Update Rollup 2 [here](https://web.archive.org/web/20100107170854/http://www.xbox.com/en-US/pcsetup/mce05lite.htm). The XEX file itself can be obtained by extracting the `mcrdinstaller/ehd.cab` file within `XboxMCELite-ENU.exe`. The XEX file is identical between language variants of the installer, i.e. the [English](https://web.archive.org/web/20070312113401/http://download.microsoft.com/download/2/8/5/28521E1E-120E-4D05-8310-3E43F2734431/XboxMCELite-ENU.exe) and [German](https://web.archive.org/web/20070128065459/http://download.microsoft.com/download/2/8/E/28E4D44A-AF98-4605-AA93-3AA958E13402/XboxMCELite-DEU.exe) executables contain the same XEX file.
